## Project Focus

The working focus is the intake-to-activation funnel: how a prospective patient moves from initial interest to completing intake and reaching the first visit.

## Core Question

Where might prospective patients face avoidable friction before completing intake or reaching the first visit, and what lightweight workflow could reduce that friction?

## Project Outputs

1. Evidence log from public funnel walkthrough
2. Review theme analysis
3. External benchmark table
4. Simple funnel impact model
5. AI-assisted workflow prototype
6. Short final report

## Guardrails

* Use only public information and synthetic sample data.
* Do not use real patient data.
* Do not make clinical recommendations.
* Label observed friction as hypotheses unless supported by data.
* Keep the project focused on growth, operations, activation, and workflow improvement.
